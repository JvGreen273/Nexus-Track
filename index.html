APP NATIVA — ELECTRON

Instala NEXUS TRACK como app nativa (.exe / .dmg / .AppImage):

  1. Instala Node.js desde nodejs.org
  2. Descarga el proyecto ZIP del repositorio
  3. Abre la carpeta en una terminal
  4. Ejecuta los comandos:
npm install npm run build:web npm run build:win # → .exe Windows npm run build:mac # → .dmg Mac npm run build:linux # → .AppImage Linux

El instalador queda en dist-electron/

INSTALAR EN iPHONE / iPAD

iOS requiere Safari para instalar. Son 3 pasos:

  1. Abre esta página en Safari (no Chrome)
  2. Toca el botón Compartir (cuadrado con flecha ↑)
  3. Selecciona "Añadir a pantalla de inicio"

Se instala como app nativa sin App Store ni costo.
