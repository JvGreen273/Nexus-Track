VERSIÓN DE ESCRITORIO — ELECTRON

Para instalar NEXUS TRACK como app nativa (.exe / .dmg / .AppImage):

  1. Descarga Node.js desde nodejs.org e instálalo
  2. Descarga el proyecto ZIP desde el botón de arriba
  3. Descomprímelo y abre la carpeta en una terminal
  4. Ejecuta los siguientes comandos:
npm install npm run build:web # Elige tu plataforma: npm run build:win # → Windows (.exe instalador) npm run build:mac # → Mac (.dmg) npm run build:linux # → Linux (.AppImage)

El instalador quedará en dist-electron/ — ejecútalo como cualquier instalador.

INSTALAR EN iPHONE / iPAD

En iOS no hay prompt automático, pero se instala en 3 pasos:

  1. Abre esta página en Safari (no Chrome)
  2. Toca el botón Compartir (cuadro con flecha hacia arriba)
  3. Selecciona "Añadir a pantalla de inicio"

La app se instalará como ícono en tu pantalla de inicio, igual que una app de la App Store. Sin costo, sin tienda.
